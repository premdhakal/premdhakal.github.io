About Project:


A web application that utilizes JavaScript, HTML, CSS with Flash Animations. Goal of the site is to make it easy and fun learning experience for students to learn CCNA 2 - Routing and Switching Basics.

Techology Used:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Adobe Animate
  • Notepad++
