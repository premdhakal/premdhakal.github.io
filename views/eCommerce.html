About Project:

eCommerce application written in Java using Spring Boot, Hibernate ORM and H2 database which utilizes JWT to handle the authorization in webservice call. THis application also demonstrates the automation of the CI/CD pipeline using tools such as Docker, Jenkins, Github and Eclipse IDE.
