About Project:

Web application written in Java using Spring Boot, Thymeleaf, MyBatis Mappers and H2 database as well as Selenium for application testing. This application also manages user access with Spring Security and also demonstrates the use of hash and encryption services. Objective of this application is to provide users store their personal files/information into a secure cloud.



Technology Used Features Implemented
  • Spring Framework
  • H2 Database
  • Thymeleaf
  • Selenium Test
  • Managing User Access with Spring Security
  • EncryptionService \ HashService
  • CRUD
