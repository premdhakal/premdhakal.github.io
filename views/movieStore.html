About Project:


This application is built inorder to fulfill the business needs for local movie stored called "MyMovie Store". Employees at MyMovie Store needed an application that can help them stay up to date with numbers of movies they have currently in store. They want to see which movies are checked-out, checked-in, due date other general tasks. Employees also wanted to be able to sell or rent movies out to customers.

Technology Used Features Implemented
  • Java
  • JFrame
  • Eclipse IDE
  • NetBeans IDE
  • Check Inventory Records
  • Sell/Rent Movies
  • Login/Logout

Application Simple and elegant application with login security features which allows employees at MyMovie Store to checkin, checkout, sell or rent movies.
